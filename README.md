# Fast Reader

Fast Reader is a minimalist mobile application designed to help users improve their reading speed and focus using Rapid Serial Visual Presentation (RSVP) techniques.

The app emphasizes clarity, concentration, and speed through a clean interface and distraction-free reading experience.
